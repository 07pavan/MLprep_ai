# Data Analyst Agent

A starter structure for a Python-based data analyst agent.

## Project Layout

- `app.py` - entry point
- `agents/` - orchestration and specialist agent modules
- `tools/` - helper utilities for data, charts, and statistics
- `utils/` - general helpers, prompts, and validators
- `config/` - application settings
- `sample_data/` - sample datasets
- `tests/` - basic test coverage

## Run

```bash
python app.py
```
